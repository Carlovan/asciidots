# val_to_id.dots

### Purpose
This librairy sets the id of a dot to its value

### Usage
    
     T
    L+R
     B

##### Parameters
- `L`, `R`, `T` or `B`: The dot to change it id
- `R`, `L`, `B` or `T`: The dot will be go out following the same direction

##### Example

    %!val_to_id.dots i

    .-#2-@6-$_@--i--$_@-$_#-&

##### Output

    666
